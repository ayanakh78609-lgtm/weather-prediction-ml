# Weather Prediction ML App 🌦️

This project predicts temperature using Machine Learning and live weather data.

## Features
- Weather prediction using ML model
- Data visualization using graphs
- Location-based weather data
- Interactive interface

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Project Files
app.py – main Streamlit application  
train_model.py – model training script  
graphs.py – data visualization  
weather.csv – dataset  
requirements.txt – dependencies
